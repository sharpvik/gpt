# `gpt` - ChatGPT 4 In Your Terminal

## Install

```bash
go install github.com/sharpvik/gpt
```

## Provide OpenAI API Key

```bash
gpt key <YOUR_OPENAI_API_KEY>
```

## Ask a Quick Question

```bash
gpt "Tell me about football"
```
